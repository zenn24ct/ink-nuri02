
### Ink Nuri02 — Color Match

色指定パズル型の Web ミニゲーム。  
指定された色を正確に塗り、制限時間内にスコアを稼ぐ。

#### ゲーム概要
- 各マスにランダムなターゲット色が割り当てられる
- パレットから色を選択してマスを塗る
- 制限時間は 60 秒
- 正解でスコア・コンボ加算
- 不正解でタイム減少（-2 秒）とコンボリセット

---

#### 操作方法
- パレットから色を選択
- Start ボタンで開始
- ヒント色と同じ色を塗る
- Hide Hints でヒント非表示に切り替え可能

---

#### 主な機能
- 8 色カラーパレット
- ターゲット色判定
- コンボ・スコア計算
- タイムアタック制
- ヒント表示切替
- ドラッグ操作対応
